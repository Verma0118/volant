## Market Timeline
- **2025–2027:** First commercial eVTOL routes (LA, NYC, Dubai)
- **2028–2030:** Fleet scaling begins — this is when fleet ops software becomes critical
- **Now:** Drone fleets are the bridge — same software, flying today

## Regulatory Landscape
- FAA Part 135 for air taxi operations
- eVTOL operators will need rigorous maintenance and compliance logging (opportunity)
- BVLOS (Beyond Visual Line of Sight) rules evolving for drones — compliance tool opportunity now

## The API / Integration Question
Critical unknown: will OEMs open up telemetry APIs?
- Samsara succeeded partly because trucks are commodity hardware + standardized OBD-II ports
- eVTOL has no equivalent standard yet
- Options: (1) build OEM partnerships early, (2) build hardware agnostic with adapter layer, (3) start with drones where API access is more open
